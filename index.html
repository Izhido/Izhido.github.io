Izhido's stuff

A collection of tools and things I have made over the years, following my own interests and hobbies.

Why?

Quite honestly, just because I can.

Also, for the giggles.

The stuff:

  • PAK: A tool to view (and extract) the contents of .pak files from custom mods & maps for Quake, the videogame created by id Software.
    Just drop a .pak file into the browser, view a list of files in it, and press the [Download] button next to them to save them directly to your computer.
  • MDL: A tool that allows you to open and view alias models (contained in .mdl files) also from Quake.
    Just drop a .mdl file into the browser to view it rendered in 3D on your screen. Use the provided controls to play all frames (and skins) stored in the model.
  • Keys: A tool that can be used to determine what keys have been pressed into the keyboard.
    Useful to test if the currently connected keyboard has any issues with multiple key pressing, for gaming or any other activities.
